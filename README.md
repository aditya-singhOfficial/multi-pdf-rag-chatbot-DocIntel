# рҹӨ– DocIntel вҖ“ Multi-PDF RAG Chatbot

A production-ready Retrieval-Augmented Generation (RAG) application that allows users to upload multiple PDF documents and interact with them through a conversational AI interface.

Built using React, Node.js, LangChain, Gemini models, and vector search, DocIntel enables accurate document-grounded question answering while minimizing hallucinations through contextual retrieval.

## вңЁ Features

- рҹ“„ Upload and process multiple PDFs simultaneously
- рҹ”Қ Semantic document search using vector embeddings
- рҹ’¬ Conversational AI chat interface
- рҹ§  Context-aware answers generated from uploaded documents
- рҹ“ҡ Source-aware retrieval for improved response accuracy
- вҡЎ Fast document indexing and retrieval pipeline
- рҹҗі Fully containerized with Docker & Docker Compose
- рҹ”’ Secure document processing and isolated storage

- **api/** вҖ“ Axios API configuration and backend communication.
- **components/** вҖ“ Reusable UI components such as chat messages and sidebar.
- **Dockerfile** вҖ“ Container configuration for frontend and backend services.
- **docker-compose.yml** вҖ“ Orchestrates the complete multi-container application.
